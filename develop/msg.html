注释提示 ​

标有 [收] 仅限收到

标有 [发] 仅限发送

标有 [选] 发送可选

特性提醒 ​

自 v4.4.14 开始 NapCat 采用 JSON5 标准解析请求，支持在 JSON 中尾随逗号与注释使用，以调用 ws/http。

json
{
  "type": "text",
  "data": {
    "text": "文本"
  }
}
